# SRS 2.0

Простой переносимый веб-сайт для интервальных повторений слов. Приложение написано на Flask и использует SQLite по умолчанию.

## Возможности
- Регистрация и авторизация пользователей.
- Боковое меню с основными разделами.
- Добавление карточек слов с примерами EN→RU и RU→EN.
- Повторение карточек со случайным выбором примеров и динамическими интервалами.
- Просмотр всех карточек, детализация и управление примерами.
- Отчёт по статистике и настройка параметров интервального повторения.

## Быстрый запуск

### Один ярлык (Windows)
1. Установите [Python 3.10+](https://www.python.org/downloads/) c галочкой «Add Python to PATH».
2. Дважды кликните по файлу `start_app.bat` в корне проекта (можно создать на него ярлык и поместить на рабочий стол).
   - При первом запуске автоматически создаётся виртуальное окружение `.venv` и ставятся зависимости.
   - Затем откроется окно, в котором будет запущен сайт. Не закрывайте его, пока работает сервер.
3. Откройте браузер и перейдите по адресу `http://127.0.0.1:5000`.

### Командная строка (Linux/macOS/Windows)
1. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```
2. Запустите приложение:
   ```bash
   python app.py
   ```
3. Откройте в браузере `http://127.0.0.1:5000`.

По умолчанию используется база данных `instance/srs.db`. Для изменения настроек используйте переменные окружения `SECRET_KEY` и `DATABASE_URI`.
